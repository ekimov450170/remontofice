# UniGroup — Сайт компании

Полный статический сайт, воссозданный с Tilda в чистый HTML/CSS/JS.

## Файлы
- `index.html` — главная страница со всем контентом
- `css/style.css` — стили
- `js/main.js` — скрипты (FAQ, форма, анимации)
- `images/` — добавьте сюда логотипы клиентов и фото работ

## Деплой на GitHub Pages
1. Settings → Pages → Source: branch `main`, folder `/`
2. Сайт: `https://<username>.github.io/<repo>/`

## Подключение домена
DNS A-записи (хост `@`):
```
185.199.108.153
185.199.109.153
185.199.110.153
185.199.111.153
```
CNAME: `www` → `<username>.github.io`

## Что нужно заменить
- Логотипы клиентов: добавьте в `/images/` и обновите секцию «Клиенты»
- Фото работ: загрузите в `/images/works/`
- Форма: настройте отправку на email или CRM (Bitrix24, AmoCRM)
